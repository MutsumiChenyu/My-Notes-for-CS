DFS深度优先搜索 和 BFS广度优先搜索

**DFS的框架**
```python
		def dfs(node):
            if not node:
                return 0
            #AAAAAAAAAAA 前序遍历   
            left = dfs(node.left)
            #BBBBBBBBBBB 中序遍历
            right = dfs(node.right)
            #CCCCCCCCCCC 后序遍历
```

- 想**传递参数**给子节点时（如：路径总和、最大值限制） -> 用 **前序**（带参数往下传）。
    
- 想**收集结果**给父节点时（如：子树高度、子树节点数） -> 用 **后序**（拿返回值往上返）。
    
- 碰到 **二叉搜索树 (BST)** 或需要 **排序** 时 -> 用 **中序**。

**本质**：前序从上往下搜索 中序从左往右搜索 后序从下往上搜索
