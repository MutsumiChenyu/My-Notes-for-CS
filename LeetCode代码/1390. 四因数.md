
#### 数学逻辑 $$一个数的因数必然只需要找到一个  而且必然有一个小于\sqrt{n} 一个大于\sqrt{ n }$$
#### 我的代码：
```python
class Solution:
    def sumFourDivisors(self, nums: List[int]) -> int:
        n = len(nums)
        Final = 0
        
        for i in range(n):
            A = nums[i]
            sum = 0
            B = 0        
            for j in range(1, int(sqrt(A)) + 1):
                if A % j == 0 and A != j*j:
                    B += 2
                    r = j + A // j
                    sum += r
                elif A % j == 0 and A == j*j:
                    B += 1
                    sum += j

            if B == 4:
                Final += sum

        return Final
```
