
### 核心算法：
奇数个负数->找到绝对值最小的负数 其他都变成正数
偶数个负数->全部变成整数

#### 我的代码
```python
	class Solution:
	    def maxMatrixSum(self, matrix: List[List[int]]) -> int:
	        Row = len(matrix)
	        Col = len(matrix[0])
	        Sum = Col * Row
	        GeShu = 0
	        Final = 0
	        Min = abs(matrix[0][0])

	        for i in range(Row):
	            for j in range(Col):
	                if matrix[i][j] < 0:
	                    GeShu += 1
	                if abs(matrix[i][j]) < Min:
	                    Min = abs(matrix[i][j])
	                matrix[i][j] = abs(matrix[i][j])
	                Final += matrix[i][j]

	        if GeShu % 2 == 0:
	            return Final
	        else:
	            return Final - Min*2
```
